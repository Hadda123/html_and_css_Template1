Un site web est l’ensemble des pages web et des ressources inter reliées
entre elles par des liens hypertextes, auxquelles l’internaute peut accéder
par une adresse web appelée Url, le tout enregistré sous le même nom de
domaine. Un site web ou site internet englobe des textes et multimédia,
il est hébergé sur un serveur web, auquel on peut accéder à travers le réseau
internet ou intranet.
